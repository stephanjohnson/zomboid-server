# Editable

Use `Editable` for inline rename or quick-edit interactions that should stay in place.

## When to Use

- Renaming rows or entities inline
- Quick edits in settings and lists

## Wraps

- `reka-ui` editable primitive

## What Shadcn Adds

- Styled preview/edit affordances aligned with other shadcn inputs

## Installation

```bash
pnpm dlx shadcn-vue@latest add editable
```

## Example Patterns

- Vanilla: inline editable text
- Contextual: editable title in a list row or card header
