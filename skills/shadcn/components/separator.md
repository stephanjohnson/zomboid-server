# Separator

Use `Separator` to visually divide related content without introducing a heavier container boundary.

## When to Use

- Split sections in cards, menus, and navs
- Inline vertical dividers in small action groups

## Wraps

- `reka-ui` separator primitive

## What Shadcn Adds

- Standard muted divider styling and orientation-friendly composition

## Installation

```bash
pnpm dlx shadcn-vue@latest add separator
```

## Example Patterns

- Vanilla: horizontal or vertical divider
- Contextual: toolbar or breadcrumb/action cluster separation

## Official Docs Example Coverage

- `Content divider`: horizontal separator between a heading block and related metadata.
- `Inline vertical divider`: small separators inside compact action groups.

## Practical Notes

- The official examples show separators as light rhythm tools, not as strong layout boundaries.
- Use vertical orientation only in tight inline layouts where spacing alone is not enough.
