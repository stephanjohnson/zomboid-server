# Date Picker

Use `DatePicker` when users should choose a date from a trigger-driven calendar surface.

## When to Use

- Form date fields with optional keyboard fallback
- Filters and scheduling controls

## Built From

- `Calendar`, `Popover`, and button/input-like trigger patterns

## What Shadcn Adds

- Documented composition of a common picker pattern with default styling

## Installation

```bash
pnpm dlx shadcn-vue@latest add date-picker
```

## Example Patterns

- Vanilla: trigger opens calendar picker
- Contextual: date filter or booking form
