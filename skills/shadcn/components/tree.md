# Tree

Use `Tree` for hierarchical data that needs nested expansion, selection, or disclosure behavior.

## When to Use

- File explorers and nested resources
- Hierarchical configuration or content structures

## Wraps

- `reka-ui` tree primitives

## What Shadcn Adds

- Styled node presentation and alignment with app-shell patterns like `sidebar`

## Installation

```bash
pnpm dlx shadcn-vue@latest add tree
```

## Example Patterns

- Vanilla: expandable tree nodes
- Contextual: file explorer or nested docs sidebar
