# Combobox

Use `Combobox` when the user needs to search or filter a list before choosing an option.

## When to Use

- Large option sets
- Searchable pickers and command-like field inputs
- Cases where `Select` is too limited for discovery

## Built From

- `Popover` + `Command` + button-like trigger patterns

## What Shadcn Adds

- A documented composition pattern for searchable selection with standard styling

## Installation

- Install `popover`
- Install `command`

## Example Patterns

- Vanilla: searchable framework picker
- Contextual: assignee picker or filter field in a form/dialog

## Related Components

- `command`, `popover`, `select`
