# Command

Use `Command` for searchable command menus, quick navigation, and palette-style interactions.

## When to Use

- Command palette overlays
- Searchable menus and quick actions
- Internal tool navigation and pickers

## Built From

- shadcn-vue command registry wrapper and supporting parts

## What Shadcn Adds

- A cohesive palette UI, input, empty state, grouping, and item styling

## Installation

```bash
pnpm dlx shadcn-vue@latest add command
```

## Example Patterns

- Vanilla: command list with search and empty state
- Contextual: dialog-based command palette or combobox composition

## Related Components

- `dialog`, `combobox`, `popover`
