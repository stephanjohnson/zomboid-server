# Chart

Use `Chart` when you need data visualization patterns that still match the shadcn surface language.

## When to Use

- Dashboards and analytics views
- KPI trend cards and comparative visualizations

## Built From

- External charting library integrations rather than Reka primitives

## What Shadcn Adds

- Chart container patterns, theme alignment, and docs-friendly examples

## Installation

```bash
pnpm dlx shadcn-vue@latest add chart
```

## Example Patterns

- Vanilla: single chart in a card
- Contextual: dashboard area or bar chart with filters
