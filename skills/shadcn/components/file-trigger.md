# File Trigger

Use `FileTrigger` patterns when a styled button or dropzone should open the system file picker.

## When to Use

- Upload buttons and media pickers
- Custom file input surfaces

## Built From

- Native file input behavior plus shadcn trigger composition

## What Shadcn Adds

- Cleaner trigger ergonomics and visual consistency with button-driven workflows

## Installation

```bash
pnpm dlx shadcn-vue@latest add file-trigger
```

## Example Patterns

- Vanilla: button opens file chooser
- Contextual: upload card or avatar uploader
