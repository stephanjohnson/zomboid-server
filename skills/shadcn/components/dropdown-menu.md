# Dropdown Menu

Use `DropdownMenu` for compact triggered action lists attached to a button or icon trigger.

## When to Use

- Row actions and account menus
- Small action bundles that do not need permanent space
- Overflow menus in toolbars and cards

## Wraps

- `reka-ui` dropdown-menu primitives

## What Shadcn Adds

- Styled menu content, items, labels, separators, shortcuts, checkbox/radio items, and submenus

## Installation

```bash
pnpm dlx shadcn-vue@latest add dropdown-menu
```

## Example Patterns

- Vanilla: trigger with menu items and separators
- Contextual: row action menu or account/user menu

## Related Components

- `context-menu`, `menubar`, `button`
