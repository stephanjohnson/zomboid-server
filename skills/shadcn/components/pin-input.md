# Pin Input

Use `PinInput` for OTP, verification code, and short multi-character token entry.

## When to Use

- Verification flows and two-factor auth
- Temporary access codes and setup steps

## Wraps

- `reka-ui` pin-input primitive

## What Shadcn Adds

- Styled per-slot boxes and shared field focus treatment

## Installation

```bash
pnpm dlx shadcn-vue@latest add pin-input
```

## Example Patterns

- Vanilla: 6-digit OTP input
- Contextual: auth or onboarding verification flow
