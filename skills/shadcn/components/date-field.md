# Date Field

Use `DateField` when users should type or step through structured date segments directly.

## When to Use

- Keyboard-friendly date entry
- Structured date input without a visible calendar

## Wraps

- `reka-ui` date-field primitive

## What Shadcn Adds

- Styled segmented input presentation that matches other shadcn fields

## Installation

```bash
pnpm dlx shadcn-vue@latest add date-field
```

## Example Patterns

- Vanilla: segmented date input
- Contextual: date field inside a form or settings row
