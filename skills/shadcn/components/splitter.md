# Splitter

Use `Splitter` for resizable multi-pane layouts where users should control space allocation.

## When to Use

- Editors, inspectors, and data workspaces
- Master-detail or multi-panel dashboards

## Wraps

- `reka-ui` splitter primitives

## What Shadcn Adds

- Styled handles and layout alignment with the design system

## Installation

```bash
pnpm dlx shadcn-vue@latest add splitter
```

## Example Patterns

- Vanilla: two resizable panes
- Contextual: editor and preview workspace
