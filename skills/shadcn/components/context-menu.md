# Context Menu

Use `ContextMenu` for alternate actions triggered by right-click, long press, or contextual action affordances.

## When to Use

- File explorer and table row actions
- Canvas, editor, or tree interactions
- Secondary actions that should stay out of the primary toolbar

## Wraps

- `reka-ui` context-menu primitives

## What Shadcn Adds

- Styled menu items, labels, separators, shortcuts, and submenus

## Installation

```bash
pnpm dlx shadcn-vue@latest add context-menu
```

## Example Patterns

- Vanilla: right-click action menu
- Contextual: file, table, or tree item contextual actions

## Related Components

- `dropdown-menu`, `menubar`, `command`
